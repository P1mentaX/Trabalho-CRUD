# 📚 Trabalho CRUD

Projeto desenvolvido durante a graduação em **Ciência da Computação**, em colaboração com o grupo formado por:  
**Eduardo Pimenta, Nael Barreto, Vinicius Ferreira e Marco Aurélio**.  

O trabalho consistiu na implementação de um sistema **CRUD (Create, Read, Update, Delete)** utilizando **Java** no **NetBeans** e **PostgreSQL** como banco de dados.

---

## 📁 Estrutura do Repositório
O repositório contém:  

- 📄 **`AtividadeFinal.zip`** → projeto completo do NetBeans  
- 📄 **`Atividade - Eduardo Pimenta - Marco Aurelio - Nael Barreto - Vinicius Ferreira.pdf`** → documentação detalhada com **MER (Modelo Entidade-Relacionamento)** e scripts SQL  
- 📄 **`postgresql-42.7.0.jar`** → driver JDBC para conexão com o banco

## 🛠️ Tecnologias Utilizadas
- **Java 8+**  
- **NetBeans IDE**  
- **PostgreSQL 17**  
- **Driver JDBC:** `postgresql-42.7.0.jar`  

---

## 🚀 Como Rodar o Projeto
1. **Descompacte** o arquivo `AtividadeFinal.zip`.  
2. Abra o projeto no **NetBeans IDE**.  
3. Configure o **driver JDBC** (`postgresql-42.7.0.jar`) no projeto, se necessário.  
4. Crie o **banco de dados** no PostgreSQL utilizando os scripts SQL fornecidos na documentação.  
5. Execute a **classe principal**: `TelaLogin` (primeira tela do sistema).
